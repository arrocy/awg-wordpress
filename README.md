# AWG WordPress

- Arrocy Whatsapp Gateway integration for Wordpress/Woocommerce and more

# Settings

- Arrocy Whatsapp Gateway URL: https://arrocy.com/api/send
- Token: Login/Register arrocy.com, add instance, copy the token.

# Templates

- Insert Admin Phone Numbers separated by commas, if you want store Admins to get notified. (optional)
- Insert Order Created Template
- Insert Order Completed Template
- Insert Order Cancelled Template
- Insert Order Refunded Template
